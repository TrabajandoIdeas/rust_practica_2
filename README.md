# Practicas Rust

> Originalmente este repo era solo para la practica 2, pero decidi usar el mismo repo para el resto de las practicas. Esto se debe a una especificacion de como organizarnos con las practicas y que considero que no valia la pena crear otro repositorio simplemento por aclarar un pequeño nombre

En este repo se encuentran todas las practicas organizadas con el siguiente criterio:
```
src
-- tpXX
  -- ejXX.rs
  -- mod.rs
-- main.rs
```

##### Leyenda:

- tpXX: Carpeta que agrupa todos los ejercicios de un unico trabajo practico, donde XX es el numero de dicho practico.
- ejXX.rs: Archivo rust de un ejercicio en especifico.
- mod.rs: Donde se organizan y exportan los modulos para su uso.
- main.rs: Archivo principal de rust.
